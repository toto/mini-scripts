mini-scripts
=====

A small collection of useful scripts and mini-tools I use from day to day

nntp2atom.rb
----------

Ruby script that turns the last 15 postings NNTP-Newsgroup into an Atom feed. 

Requires:

 - ruby >= 1.8.6
 - gems
    - builder
    - nntp
